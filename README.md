# Secure Password Vault

## Project Description

Secure Password Vault is a PHP OOP and MySQL web application for storing encrypted password records.

## Planned Features

- User registration and login
- Hashed login passwords
- Permanent encrypted user key
- Encrypted website/program password storage
- Automatic date and time storage
- Password generator
- Change login password function
- UML class diagram
- Database diagram
- Test report with screenshots

## Technologies

- PHP
- MySQL / phpMyAdmin
- HTML and CSS
- Git and GitHub
- XAMPP
